# 浏览器插件Hello-World

创建`manifest.json`并且在内部放入以下代码

```
{
  "name": "Getting Started Example",
  "description": "Build an Extension!",
  "version": "1.0",
  "manifest_version": 3
}
```
在chrome中打开这个链接
`chrome://extensions`
